# Components Docs

- `components/external-link.tsx`: Cross-platform external link wrapper for opening URLs.
- `components/haptic-tab.tsx`: Tab bar button with haptic feedback.
- `components/hello-wave.tsx`: Simple animated wave icon demo component.
- `components/parallax-scroll-view.tsx`: Scroll view container with parallax header behavior.
- `components/themed-text.tsx`: Text component with theme-aware color variants.
- `components/themed-view.tsx`: View component with theme-aware background color.
- `components/ui/collapsible.tsx`: Expand/collapse container with animated content visibility.
- `components/ui/icon-symbol.tsx`: Cross-platform symbol icon mapping wrapper.
- `components/ui/icon-symbol.ios.tsx`: iOS native SF Symbols implementation.
- `components/ui/floating-action-button.tsx`: Reusable floating action button with configurable icon and colors.
- `components/outline/outline-loading-skeleton.tsx`: Skeleton placeholders for outline loading state.
- `components/outline/outline-item.tsx`: Single outline item renderer with interaction support.
- `components/outline/outline-item-list.tsx`: List renderer for outline items.
- `components/outline/outline-keyboard-toolbar.tsx`: Keyboard accessory toolbar for outline editing actions.
